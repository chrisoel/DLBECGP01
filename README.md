# Glossar - Softwaretest

Dieses Projekt ist ein webbasiertes Glossar-Prototyp für die Log7-Consult GmbH, entstanden im Rahmen der Fallstudie des Moduls `DLBECGP01`. Die Anwendung wurde als statische Multi-Page-Webseite mit HTML, CSS und JavaScript umgesetzt und bietet eine Startseite, fünf fachlich verknüpfte Begriffsseiten, eine clientseitige Suche sowie eine zentrale Pflege der Glossardaten über `begriffe.json`.

Ziel des Projekts ist ein schneller, strukturierter Zugriff auf verständliche Erklärungen zentraler Fachbegriffe aus dem Softwaretest bei gleichzeitig einfacher Wartbarkeit und Erweiterbarkeit.
